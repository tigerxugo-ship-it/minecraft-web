## Phase 2 技术实现

### 昼夜循环
- 10分钟一个完整周期
- 天空颜色渐变: 白天(天蓝) → 日落(橙红) → 夜晚(深蓝) → 日出(金黄)
- 1800颗星星粒子系统
- 太阳/月亮弧形轨迹

### 工具系统
- 5种材料: 木、石、铁、金、钻石
- 挖掘速度倍率: 金12x > 钻石8x > 铁6x > 石4x > 木2x
- 耐久度系统和损坏机制
- 工具对特定方块有效

### 树木生成
- 橡树: 椭球形树冠，高度4-6
- 云杉: 锥形分层树冠，高度6-10
- 白桦树: 较窄树冠，高度5-7
- 树叶腐烂系统 (4-8秒)

### 生物AI
- 猪: 10HP，速度2.5
- 牛: 10HP，速度2.0
- 羊: 8HP，速度2.2，多色羊毛
- AI状态机: IDLE → WANDER → FLEE
- 行走动画和血条显示
