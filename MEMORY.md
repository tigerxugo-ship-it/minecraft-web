# Minecraft Web 项目记忆

## 项目概述
基于 React + Three.js 的网页版 Minecraft 克隆项目。

---

## Phase 3 功能验证 (2026-02-16)

### 测试状态: ✅ 通过

### 完成的功能模块

#### 1. 🧟 敌对生物系统
- **僵尸**: 追逐AI、阳光下燃烧、掉落系统
- **骷髅**: 远程射箭、保持距离行为
- **苦力怕**: 充能爆炸机制、视觉效果
- **蜘蛛**: 攀爬能力、白天中立行为
- 所有敌对生物均有完整的AI状态机

#### 2. ⛏️ 制作系统
- 3x3 有序配方匹配
- 快捷合成 (5种常用配方)
- 熔炉冶炼系统 (铁/金矿石 → 锭)
- 燃料系统 (煤炭80秒/木头15秒)
- 工具合成 (木/石/铁/金/钻石镐和剑)

#### 3. 💎 矿石生成系统
- 煤矿石 (0-128层, 每区块20个矿脉)
- 铁矿石 (0-64层, 每区块20个矿脉)
- 金矿石 (0-32层, 每区块2个矿脉)
- 钻石矿石 (0-16层, 每区块1个矿脉)
- 基于种子的确定性生成算法

#### 4. 💾 存档系统
- IndexedDB 持久化存储
- RLE 压缩算法
- 自动保存到 LocalStorage
- 导入/导出功能
- 存档管理UI

### 构建结果
```
✅ TypeScript 编译通过
✅ Vite 构建成功
✅ 输出文件完整 (index.html + assets/)
```

### 技术栈
- React 18.2.0
- TypeScript 5.2.2
- Three.js 0.160.0
- Zustand (状态管理)

### 项目结构
```
src/
├── entities/    # 生物AI (敌对+被动)
├── crafting/    # 制作系统
├── world/       # 世界生成 (矿石)
├── save/        # 存档系统
├── tools/       # 工具系统
├── blocks/      # 方块定义
└── engine/      # 游戏引擎
```
